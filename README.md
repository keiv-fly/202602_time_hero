# time-hero

Turn-based browser game prototype built with PixiJS, TypeScript, and Vite.

## Prerequisites

- Node.js 18+ (recommended: latest LTS)
- npm (comes with Node.js)

## Install dependencies

```bash
npm install
```

## Run the game locally

Start the development server:

```bash
npm run dev
```

Then open:

- `http://localhost:5173`

## Run tests

Run the test suite once (Vitest):

```bash
npm run test
```

## Build for production

```bash
npm run build
```
